# Datapack-PVPText
一个MinecraftJava的数据包，能够显示一些简单的“提示”

-------------------------------------------------------------
> 此数据包函数**及其屎山**......

- 此数据包添加了一些与PVP相关的提示或反馈，比如：
- 更改了杀死玩家或玩家死亡的提示（但原来的死亡提示没了ing...）
- 玩家死亡后会变为旁观模式并强制旁观玩家（1.3版本以上）
- 显示击杀、死亡次数和场地剩余玩家数：[75/100]
- MinecraftJava数据包开发真的好小众...如果有爱好者一起的话...谢谢！
